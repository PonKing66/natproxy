# Getting started

1. Download
   ```shell
   git clone https://github.com/PonKing66/natproxy
    ```
2. Build
   ```shell
   cd natproxy
   mvn clean package -Dmaven.test.skip=true
    ```
3. Run
   1. 
   ```shell
   cd ../appassembler/natproxy
    ```
   2. run client on the proxy client.
   3. run server on the proxy server.
